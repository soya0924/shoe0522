# AI 智慧步數分析系統

## 專案說明
這是一個結合 AI 技術的智慧步數分析系統，通過加速度感測器收集使用者的步數數據，並利用 AI 演算法分析每日運動量，提供個人化的運動建議。

## 功能特色
- 精確步數統計
- 每日運動量追蹤
- AI 運動建議生成
- 數據視覺化展示
- 長期趨勢分析

## 系統架構
### 硬體需求
- Arduino Nano（體積小、便於安裝）
- MPU6050 加速度計（用於步數統計）
- 鋰電池（3.7V）
- 充電電路
- HC-05 藍牙模組
- 簡易鞋墊底座

### 軟體架構
1. **前端網頁介面**
   - React.js 前端框架
   - Chart.js 數據圖表（步數統計視覺化）
   - Material-UI 介面組件
   - PWA 支援離線使用

2. **後端系統**
   - Node.js/Express 伺服器
   - MongoDB 資料庫（儲存步數和運動記錄）
   - AI 模型整合（TensorFlow.js，用於運動建議）
   - RESTful API

3. **Arduino 控制系統**
   - 步數感測與計算
   - 藍牙數據傳輸
   - 低功耗管理
   - 簡單數據處理

### 3. 軟體開發（3週）
- Arduino 程式開發
  - 步數計算演算法
  - 數據傳輸程式
  - 省電模式管理

- 前端網頁開發
  - 每日步數展示
  - 運動量統計圖表
  - 個人目標設定
  - 運動建議顯示

- AI 模型開發
  - 日常運動量分析
  - 個人化建議生成
  - 運動目標調整

## 資料分析指標
1. **步數分析**
   - 每日步數統計
   - 活動時段分布
   - 達標率追蹤

2. **運動建議**
   - 根據年齡和體能建議每日步數
   - 根據天氣調整運動建議
   - 根據歷史數據調整運動強度
   - 提供適合的運動方式建議

3. **趨勢分析**
   - 週/月步數統計
   - 運動習慣分析
   - 目標達成進度

## 安裝與執行
```bash
# 1. 克隆專案
git clone [repository_url]

# 2. 前端設定
cd frontend
npm install
npm start

# 3. 後端設定
cd ../backend
npm install
cp .env.example .env
npm start

# 4. Arduino 程式
# 使用 Arduino IDE 開啟 /arduino/smart_insole.ino
# 編譯並上傳至 Arduino Nano
```
