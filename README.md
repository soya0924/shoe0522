# AI 智慧步數分析系統

## 設計動機
現代許多年輕人與上班族因工作忙碌與生活壓力，難以固定安排運動時間，導致日常身體活動量不足，影響健康狀況。為了改善這個問題，系統將透過智慧型手機或穿戴式裝置即時記錄使用者每日步數，並根據其活動量，提供個人化的運動建議，鼓勵持續行動。當使用者達成設定的目標步數時，系統也會給予虛擬徽章、健康積分或其他小獎勵，藉此提升持續運動的動力。透過這樣的方式，幫助使用者在日常生活中逐步建立起規律運動的習慣，進一步促進健康與生活品質。

## 專案說明
這是一個結合 AI 技術的智慧步數分析系統，通過加速度感測器收集使用者的步數數據，並利用 AI 演算法分析每日運動量，提供個人化的運動建議。系統支援自訂步數目標，並在達標時提供互動式的獎勵動畫。

## 功能特色
- 精確步數統計
- 每日運動量追蹤
- AI 運動建議生成
- 數據視覺化展示
- 長期趨勢分析
- 個人化步數目標設定
- 達標獎勵動畫
- 成就系統

## 系統架構
### 硬體需求
- Arduino Nano（體積小、便於安裝）
- MPU6050 加速度計（用於步數統計）
- 鋰電池（3.7V）
- 充電電路
- HC-05 藍牙模組
- 簡易鞋墊底座

### 軟體架構
1. **前端網頁介面**
   - React.js 前端框架
   - Chart.js 數據圖表（步數統計視覺化）
   - Material-UI 介面組件
   - PWA 支援離線使用
   - Three.js 3D獎勵動畫
   - Lottie 動畫效果
   - LocalStorage 目標儲存

2. **後端系統**
   - Node.js/Express 伺服器
   - MongoDB 資料庫（儲存步數和運動記錄）
   - AI 模型整合（TensorFlow.js，用於運動建議）
   - RESTful API
   - 用戶目標管理系統
   - 成就追蹤系統

3. **Arduino 控制系統**
   - 步數感測與計算
   - 藍牙數據傳輸
   - 低功耗管理
   - 簡單數據處理
